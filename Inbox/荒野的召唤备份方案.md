
完整操作流程之适合在Linux系统下执行

# 1. 创建备份目录

> 只需要执行一次

```bash
mkdir -p ~/cotw_backup
```

# 2. 将整个存档文件夹压缩打包

> 自行检查游戏存档目录位置

```bash
tar -czvf ~/cotw_backup/cotw_save_$(date +%Y%m%d).tar.gz ~/.var/app/com.valvesoftware.Steam/.local/share/Steam/steamapps/compatdata/518790/pfx/drive_c/users/steamuser/Documents/Avalanche\ Studios/COTW/Saves/76561199790508092
```

# 3. 用文件管理器打开

> 需要在指定目录下操作

```bash
nautilus .
```
